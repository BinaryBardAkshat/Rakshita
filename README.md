# Rakshita AI - Road Safety Assistant

Rakshita AI is an AI-powered assistant developed as part of the Rakshak project, aimed at saving people's lives from road accidents. This project utilizes various technologies such as machine learning, computer vision, and IoT to improve road safety and assist in accident prevention.

## Overview

Rakshita AI incorporates several modules to achieve its objectives:

1. **Voice Assistant Module**: Trained using machine learning techniques to recognize voice commands and provide real-time assistance to drivers and passengers.

2. **Number Plate Detection Module**: Utilizes computer vision techniques to detect and recognize vehicle license plates for identification and tracking purposes.

3. **Distance Measurement Module**: Measures the distance between vehicles using sensors to provide warnings about safe following distances and potential collisions.

4. **ECU Mapping Module**: Retrieves information from the vehicle's Electronic Control Unit (ECU) to monitor and analyze vehicle performance and behavior.

5. **Data Transmission Module**: Sends collected data, including voice commands, number plate information, distance measurements, and ECU data, to a web server for analysis and storage.

## Repository Structure

- 📞 call.st.1.py: Python script to train the voice assistant model using machine learning techniques.
- 🚗 ECU.py: Python script for detecting and recognizing vehicle license plates using computer vision.
- 🔒 encryption.py: Python script for measuring the distance between vehicles using sensors.
- 🔍 finder.py: Python script to retrieve information from the vehicle's Electronic Control Unit (ECU).
- 📄 main.jsx: Python script to send collected data to a web server for analysis and storage.
- 🤖 rakshita.py: Python script to run all modules simultaneously.
- 📝 README.md: This file, providing an overview of the Rakshita AI project.
- 📡 reciver.ino: Text file listing the required Python packages for running the scripts.
- 📋 requirements.txt: Instructions for setting up and using the Rakshita AI system.
- ✉️ Sender.py: Python script to send collected data to a web server for analysis and storage.
